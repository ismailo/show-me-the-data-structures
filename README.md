# Show Me the Data Structures

This repository contains solutions for the Show Me the Data Structures project. Each problem demonstrates how core data structures can be used to solve practical programming problems efficiently.

## Project Structure

- `problem_1.py` to `problem_6.py`: Python solutions for each problem
- `explanation_1.md` to `explanation_6.md`: explanation of approach, time complexity, and space complexity
- `testdir/`: sample test directory used for file-based problems
- `.gitignore`: ignores unnecessary system files

## Topics Covered

This project applies common data structures and problem-solving techniques such as:

- Linked lists
- Hash maps / dictionaries
- Trees
- Recursion
- Stacks and queues
- Searching and sorting concepts
